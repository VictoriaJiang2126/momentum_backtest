# Russell 3000 动量回测系统

基于 Russell 3000 的量化动量策略回测框架，包含两套独立策略：**短线动量策略**（主策略）和**趋势轮动策略**（含 TQQQ 机制）。回测期间为 2024–2025 年，初始资金 $100,000。

## 策略一：短线动量策略（主策略）

### 核心逻辑

每日扫描 Russell 3000 全量股票，寻找处于上升趋势且近期出现放量突破的标的，次日开盘买入，持有至止损或趋势破坏。

### 入场条件（信号日前一收盘，次日开盘执行）

| 条件 | 参数 |
|------|------|
| 股价高于最低价格要求 | ≥ $5 |
| 20日均量过滤 | ≥ 500,000 股/日 |
| 三均线多头排列 | 收盘 > SMA20 > SMA50 > SMA200 |
| 接近 52 周新高 | 距最高价回撤 ≤ 10% |
| 近5日（周）涨幅 | 10% ~ 30%（动量适中，排除过热） |
| 相对成交量（RVol） | ≥ 2.0 倍均量 |
| 市场过滤器 | SPY 收盘 > SPY 200日均线 |

### 候选排序

满足入场条件的股票按 **周涨幅 × min(RVol, 5)** 综合评分降序排列，截断 RVol 上限为 5 倍，防止事件驱动个股爆炸性成交量扭曲评分。

### 出场条件（任一触发则次日开盘卖出）

| 出场规则 | 参数 |
|----------|------|
| 硬止损 | 跌破买入价 7% |
| 移动止盈 | 从持仓期最高价回撤 10% |
| 趋势出场 | 收盘跌破 SMA10 |
| 最大持仓时间 | 30 个交易日 |

### 仓位管理

- 最多同时持有 **5 只**股票
- 每只仓位为当时净值的 **20%**
- 手续费 **0.01%**（双边）

## 策略二：趋势轮动策略（TQQQ 切换）

### 核心创新：双模式切换

策略引入"市场领跑股"检测机制，在**动量活跃**和**低波动**两种市场状态间自动切换：

| 市场状态 | 判断标准 | 策略行为 |
|----------|----------|----------|
| 动量活跃 | ≥ 3 只大市值股 3 个月涨幅 > 50% | 持有高动量个股（最多5只） |
| 低波动 / 过渡期 | 领跑股不足 3 只且 SPY > 200日线 | 全仓切换 TQQQ |
| 熊市 | SPY < 200 日均线 | 清空所有持仓，空仓等待 |

### 入场条件（趋势策略）

| 条件 | 参数 |
|------|------|
| 股价 | ≥ $10 |
| 日均成交量 | ≥ 100 万股 |
| 日均成交额 | ≥ $3 亿 |
| 价格位置 | 收盘 > SMA50 > SMA200，且收盘 > SMA200 |
| 3 个月动量 | > 20% |
| 偏离 SMA50 | > 3%（避免刚刚突破后回踩） |
| 市场过滤 | SPY > 200日均线 |

候选按 3 个月动量降序排列，每周一调仓，持有动量前 5 名（容忍度：前 10 名内保留不换手）。

### 出场条件（趋势策略）

| 出场规则 | 触发条件 |
|----------|----------|
| 单日暴跌熔断 | 当日跌幅 ≥ 15% |
| 两日暴跌熔断 | 两日累计跌幅 ≥ 15%（持仓 ≥ 2 天） |
| 三日暴跌熔断 | 三日累计跌幅 ≥ 15%（持仓 ≥ 3 天） |
| 趋势出场 | 收盘跌破 SMA50 |
| 周度亏损熔断 | 当周净值回撤超阈值，暂停当周新建仓 |
| 周调仓换手 | 每周一：不在动量前 10 名的持仓清出 |
| 机制切换 | 市场状态变化时强制清仓 |

## 防前视偏差设计

- **信号日与成交日分离**：t 日收盘后生成信号，t+1 日开盘价执行买卖，严格杜绝前视偏差
- **指标使用前一日数据**：出场判断用 `prev`（前一日收盘），成交用 `today`（当日开盘）
- **预热期**：数据从 2023-01-01 开始下载，回测从 2024-01-01 起算，确保 SMA200、52 周高点等长周期指标充分预热

## 关键局限性

**幸存者偏差（最重要）**
回测仅使用当前 Russell 3000 成分股，已退市、并购或破产的股票不包含在内。这会系统性高估历史收益约 2–5%/年，真实表现会更低。

**回测区间较短**
目前回测仅覆盖 2024–2025 年，恰逢 AI 牛市主升段，样本偏好。完整评估应覆盖 2018Q4 暴跌、2020 新冠崩盘、2022 熊市等多个市场周期（建议至少 8–10 年）。

**最大回撤偏高**
-60.63% 的最大回撤主要由 TQQQ 三倍杠杆在高波动期的剧烈波动导致，实际可接受程度取决于风险偏好。

**滑点未建模**
仅扣除 0.01% 手续费，未考虑实际冲击成本（尤其是小市值股票，流动性较差时买卖价差可能显著）。

**参数过拟合风险**
所有阈值（止损比例、RVol 阈值、动量窗口等）均在同一数据集上调参，存在过度拟合短期市场特征的风险，建议用样本外数据验证。

## 技术实现说明

- **信号计算**：全向量化（pandas），矩阵运算，速度快
- **回测引擎**：自定义事件驱动日循环，非 vectorbt，精确控制成交逻辑
- **数据缓存**：增量下载，读取已有 parquet 后仅补缺失 ticker，每 20 批写一次磁盘防丢失
- **股票池**：从 iShares IWV ETF CSV 实时抓取，缓存到 `cache/russell3000.parquet`
